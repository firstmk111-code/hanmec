텍스트 관리

전체 텍스트 관리

사이트 각 페이지의 노출 텍스트(상단 제목·서브 문구)를 카테고리 / 페이지별로 수정합니다. 텍스트를 비워 두면 기존 HTML 기본값이 그대로 표시되며, 저장 시 실제 index.html에 반영됩니다. (저장 전 자동 백업)

선택 페이지 일괄 서체 조정 (현재 필터된 블록에 값 적용 — 빈 칸은 변경 안 함)

※ 폰트 종류·색상·레이아웃은 변경하지 않습니다. 크기 10~100px, 자간 -5~10px, 행간 1.0~2.5, 여백 0~100px 범위로 안전 제한됩니다. 「미연결」로 표시된 블록은 현재 페이지 구조에서 앵커를 찾지 못한 항목으로, 저장해도 프론트에 영향을 주지 않습니다.

이미지 교체

사이트에 삽입된 모든 이미지(로고·히어로 슬라이드·오버뷰·제품·아이콘 등)입니다. 교체할 이미지를 선택해 새 파일을 올리면 실제 index.html에 즉시 반영됩니다.

배너 영상 관리

메인 상단 배너 슬라이드의 첫 번째로 영상을 추가합니다. 기존 이미지들은 그 뒤 슬라이드로 유지되어 5초마다 순서대로 전환됩니다. 기능을 끄면 기존 이미지 슬라이드만 그대로 표시됩니다. (기본값: 꺼짐 · 저장 전 자동 백업)

※ 영상 파일을 사이트 폴더(예: videos)에 넣은 뒤 그 경로를 입력하세요. 경로는 반드시 / 로 시작하는 웹 경로여야 하며, C:\ ... 같은 윈도우 파일 경로는 사용할 수 없습니다. 브라우저 호환을 위해 MP4(H.264) 형식을 권장하며(.mov는 재생이 안 될 수 있음), 용량이 크면 페이지가 느려지니 웹용으로 압축하세요. 모바일 자동재생을 위해 영상은 항상 음소거·자동재생됩니다.

※ 배너의 디자인·문구·전환 방식은 그대로 유지되며, 영상이 슬라이드 목록의 맨 앞에 추가될 뿐입니다. "배너 영상 사용"을 끄고 저장하면 프론트는 기존과 100% 동일하게 복원됩니다.

하단바 · 회사정보 관리

웹사이트 하단(푸터)의 회사 정보·대표 전화·저작권 문구를 수정합니다. 저장하면 실제 index.html 푸터에 즉시 반영됩니다. (저장 전 자동 백업)

회사 정보

※ 하단바 디자인·레이아웃은 그대로 유지되며, 입력한 텍스트 내용만 안전하게 교체됩니다.

팝업 · 공지 관리

사이트 접속 시 노출되는 팝업을 운영자가 직접 켜고 끌 수 있습니다. 비활성(기본값)이면 프론트 화면은 기존과 100% 동일하며, 활성화한 기간에만 팝업이 표시됩니다.

팝업 설정

저장 시 index.html의 관리 전용 블록에만 팝업이 주입됩니다. 비활성화하면 블록이 비워져 흔적이 남지 않습니다.

공지 목록

공지 데이터를 등록·관리합니다. (백업에 포함되며, 프론트 공지 페이지 연동은 추후 확장 대상)

SEO 관리

사이트 공통 SEO·페이지별 메타·구조화 데이터·이미지 ALT·검색엔진 파일을 관리합니다.
